参加 PHP 开发者大会有感

2016.05.14 xdd1874 生活

昨晚转发了微博抽中了第二届 PHP 全球开发者大会的门票，今天就去参加了，收获还是蛮大的。写篇博文记录下今天的一些感想。

1.

说说韩天峰老师吧，Swoole项目的创始人，PHP语言官方扩展开发组成员。已经拥有各种很牛逼的title了，但是本尊非常低调谦虚。韩老师本人要比照片看上去更清秀、稳重，说话不紧不慢，温文尔雅。有一股儒家气息，讲了很多关于Swoole的应用与实践，然而技术太low的我并没有听得很懂(๑• . •๑)。

2.

今天也是如愿见到了鸟哥惠新宸ヽ(✿ﾟ▽ﾟ)ノ。偶像级的鸟哥自然而然也是做了很棒的分享，讲了 PHP 7 高性能之源。鸟哥虽然是国内PHP第一人，可是人家是写 C 的，通篇都在讲一些 C 语言的东西，带着大家体会了一下 PHP 5 到 PHP 7 zval变量容器变化中的一些思想。这么高深的东西，我当然再次并没有听懂(๑• . •๑)。但是感受到了鸟哥的一些自带 buff, 和蔼可亲，幽默诙谐，没有任何大神的架子，演讲过程也很能活跃现场气氛，现场回答问题，不管是低级的技术问题还是与技术无关的问题，鸟哥都会知无不言，耐心讲解。

提问环节当然也是异常火爆，大家都难以抑制对鸟哥的崇拜之情。从鸟哥的回答当中，能感受到鸟哥其实也是一个很普通的一名开发者，喜欢打 DOTA, 喜欢喝酒，脾气偶尔也会暴躁，但是是真的很喜欢写代码。鸟哥能走到今天，应该和他以技术为乐趣是分不开的。他很喜欢挑战 BUG, 一遇到棘手的问题就会很兴奋，特别是别人解不开他却能解开的 BUG, 那种成就感真的是最美的感觉。总之，鸟哥真的是一位非常值得我学习的牛人！

3.

今天现场也来了一个盲人程序员，他的网名叫晴天。自学编程多年，平时都是拿记事本写代码，然后用软件（好像是他自己写的）读出他敲击的每一个字符。不得不说很励志！他主要说了一下关于无障碍服务。因为现在很多软件应用对于盲人残疾人很不友好，他希望开发者们能够考虑一下盲人群体，能让他们也能够更好的体验互联网带来的便利。

4.

最好语言的大会必然也是最好的，现场可以领取各种免费的礼品，总而言之能够参加这次大会真的很开心ヽ(✿ﾟ▽ﾟ)ノ
